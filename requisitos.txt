Proyecto de consolidación de Antonio Bayá Castells (Grupo 4):

Las librerías usadas en el proyecto son:

request, para descargar datos de la web, versión 2.32.3

pandas, para manipulación de datos, versión 2.2.3

mysql.connector, para conectar con una base sql, versión 9.1.0

sqlalchemy para interactuar con datos relacionales, versión 2.0.34

jsonlines, para crear archivo JSON, versión 4.0.0

matplotlib.pyplot, para gráficos, versión 3.10.0

seaborn, para gráficos, versión 0.13.2
